Module développement web incontournable dans la création d'un site internet


    Découverte et apprentissage de deux langage web
  • Le Html :

    • Le HTML ou Hypertext Markup Lanuage n'est pas un langage web de programation. Il s'agit plutôt d'un longage de balisage du contenu d'une page sur un site internet. les blises spécifiques utilisées dans le langage HTML permettent d'inclure certains fichiers tels que les feuilles de style,les images ou les fichiers JavaScript. Ansi, le HTML, permet egalement de donner un forne générale au texte qui sera affiché sur la page de votre site web.

  • CSS :

    • A l'instar du langage HTML,le CSS ou feuille de style en cascade facilite la mise langage web de programation. Il s'agit plutôt d'un longage de balisage du contenu d'une page sur un site internet. les blises spécifiques utilisées dans le langage HTML permettent d'inclure certains fichiers tels que les feuilles de style,les images ou les fichiers JavaScript. Ansi, le HTML, permet egalement de donner un forne générale au texte qui sera affiché sur la page de votre site web.
